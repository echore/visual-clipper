stay hidden if a file cannot load, so missing shots never break the page.

- `obsidian-reveal-vault.png`: right-click menu on a note with Show in system explorer visible

- `obsidian-enable-plugin.png`: Community plugins settings with Vault Autopilot toggled on
- `obsidian-gallery.png`: gallery view with the ribbon icon and a card wall of 3+ collected videos
- `obsidian-settings.png`: Vault Autopilot settings tab with the folder and SOP sections in frame
- `obsidian-ai-install.png`: a local AI agent session (for example Claude Code) completing the install from the provided prompt
