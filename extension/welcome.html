东西存在哪

零配置即可用。一个视频 = 一条笔记：对同一个视频先后点封面、Hook、关键帧， 内容都追加进同一条笔记，点几个存几个，顺序随意。想换位置：Obsidian → 设置 → Vault Autopilot → 存储位置。

四种模式

截图

框选任意区域：点工具栏图标 → 截图 → 在页面上拖拽选区 → 存为截图笔记。

封面

收藏封面：在 YouTube / Bilibili 视频页点收藏封面 → 抓封面图 + 标题元数据 → 存入视频笔记。

Hook

抓取视频开头：在视频页点 Hook 分析 → 自动采样开头到当前播放位置的候选帧 → 网格里挑选要保留的帧 → 连同字幕存入视频笔记。

关键帧

标记 In / Out：在视频页点关键帧 → 侧边栏标记起止点 → 网格里挑选要保留的帧 → 存入视频笔记。

提示

· 保存成功后会自动跳转到 Obsidian 里对应的笔记（首次会弹「打开 Obsidian」，可勾选「始终允许」之后免确认）。

· chrome:// 等受浏览器限制的页面无法框选，截图模式会自动改存整页并在弹窗里提示。

· 使用时请保持 Obsidian 开着——扩展通过本地端口把内容发给插件，Obsidian 没开就发不进去。

高级：端口（仅端口冲突时需要改）

扩展与插件通过本地端口通信，默认 17183。 若插件提示端口被占用且你无法关闭占用程序，在这里和插件设置（高级 → 端口）改成同一个新端口。
