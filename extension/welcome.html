Screenshot Clipper

看到好设计 → 框选 → 自动存进 Obsidian，AI 填好客观分析

第一步：安装本地 Host

1

在终端里进入项目目录，运行安装脚本：

cd ~/Documents/fang/screenshot-clipper
bash install.sh

脚本会问你 Extension ID（见第二步）。

第二步：复制 Extension ID

2

你的 Extension ID 是：

（加载中…）

把它粘贴给 install.sh 的提示。

⚠️ 每次重新"Load unpacked"都会生成新 ID，需要重新运行 install.sh。上架 Chrome Web Store 后 ID 才会固定。

第三步：设置 Vault 名称

3

点击工具栏的插件图标，在弹窗里填入你的 Obsidian Vault 名称。

Vault 名称必须和 Obsidian 标题栏里显示的完全一致（大小写、空格都算）。

使用方法

▶

在任何网页上，点击工具栏图标 → 点"Clip 当前页面" → 在页面上拖拽选择你想截的区域 → 松手 → 等几秒钟 → 笔记自动出现在 Obsidian 里。
